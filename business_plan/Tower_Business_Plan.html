Subscription Model

You provide the AI, users pay monthly. Simple value proposition.

Tier Price Features
Free $0 Basic Tower (no AI), 3 active items, manual entry only
Pro $6.99/mo or $59/yr AI capture, unlimited items, habit sync, insights
Annual Discount $4.99/mo effective 2 months free (common industry practice)

Pros

  • Predictable revenue
  • Users don't need to manage API keys
  • Higher perceived value

Cons

  • You absorb AI costs (margin risk)
  • Heavy users can cost more than they pay
  • Requires hitting pricing sweet spot

Bring Your Own Key (BYOK)

Users provide their own Claude API key. You build the interface, they pay Anthropic directly.

Tier Price Features
Free (BYOK) $0 Full features, user provides API key
Tip Jar / Donations Optional Support development

Pros

  • Zero AI cost to you
  • Appeals to power users who already have API keys
  • No margin concerns

Cons

  • High friction: users must get API key
  • Limits addressable market significantly
  • Hard to monetize beyond donations
  • Most ADHD users won't complete key setup

Warning: BYOK works well for developer tools (JetBrains, VS Code, Warp) where users are technical. For consumer ADHD apps, this is likely too much friction.

Hybrid Model (Recommended)

Best of both worlds: easy entry for most users, power option for technical users.

Tier Price Features
Free $0 Basic Tower, 5 AI calls/day, manual entry
Pro $6.99/mo Unlimited AI (you pay), all features
Pro BYOK $2.99/mo All features, user provides API key

Why This Works

  • Free tier demonstrates value (limit prevents abuse)
  • Pro tier is simple for most users
  • BYOK tier captures technical users who want control
  • BYOK users still pay (you provide value beyond API)

Risk Analysis

Platform Risk: Apple Sherlock

Medium-Low Risk

Apple's Focus Modes are generic. ADHD-specific features (brain dump parsing, attention steering, 2-min timer psychology) are too niche for Apple to build.

Mitigation: Differentiate on ADHD-specific psychology, not just features.

AI Cost Escalation

Medium Risk

Heavy users could cost more in API fees than they pay in subscription.

Mitigation: Use Haiku for parsing (cheapest), implement rate limits on free tier, offer BYOK for power users.

Competition from Inflow/Todoist

Medium Risk

Established players have more resources and users.

Mitigation: Focus on "attention steering" positioning (unique), not general task management.

Low Retention

Medium Risk

Productivity apps have notoriously low retention. ADHD users may try and abandon.

Mitigation: Design for ADHD brains specifically (less overwhelming, trust the algorithm).

App Store Rejection Risk

Low Risk Tower doesn't violate any obvious guidelines. Apps with AI features are common. No medical claims = no regulatory issues.

Sources: Inflow, iGeeksBlog
